    MetricsLib.Telemetry (metrics_lib v0.1.0)

    Copy Markdown

    Attaches handlers to :telemetry events emitted by Phoenix and Ecto.

    Call attach_defaults/1 once in your application's start/2, or pass an explicit list of event specs to attach/1 for fine-grained control.

    Example

    # application.ex
def start(_type, _args) do
  MetricsLib.Telemetry.attach_defaults(repo: MyApp.Repo)
  # ...
end
